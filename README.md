# 🗳 Election Assistant Chatbot

## 📌 Project Overview
This project is an interactive **Election Assistant Chatbot** designed to help users understand the election process, timelines, voting steps, and eligibility in a simple and user-friendly way.

The application provides a chatbot-like interface where users can ask questions or use quick buttons to get information instantly.

---

## 🎯 Features
- 💬 Chatbot-style user interface
- ⚡ Quick reply buttons for fast navigation
- 🧠 Keyword-based intelligent response system
- 🎨 Modern UI with glassmorphism and AI-themed background
- ⌛ Typing animation for better user experience
- 📱 Responsive and easy-to-use design

---

## 🧠 Approach & Logic
The assistant uses a **keyword-based decision system** to simulate intelligent conversation.

- User input is captured and processed
- The system checks for keywords like:
  - "process"
  - "timeline"
  - "vote"
  - "eligibility"
  - "india"
- Based on input, the chatbot returns structured responses

This approach ensures:
- Fast performance
- Simple implementation
- Logical interaction flow

---

## ⚙️ How It Works
1. User enters a query or clicks a button
2. JavaScript processes the input
3. Matching response is generated
4. Chatbot displays reply with typing effect
