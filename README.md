# 🗳 Election Assistant Chatbot

## 📌 Project Overview
This project is an interactive AI-style **Election Assistant Chatbot** designed to simplify and explain election processes, timelines, voting steps, and eligibility through an intuitive conversational interface.

The application provides a chatbot-like experience where users can ask questions or use quick buttons to get information instantly in an easy-to-understand way.

---

## 🌐 Live Demo
You can run this project locally by opening the `index.html` file in any web browser.

---

## 🎯 Features
- 💬 Chatbot-style user interface
- ⚡ Quick reply buttons for fast navigation
- 🧠 Keyword-based intelligent response system
- 🎨 Modern UI with AI assistant visual design
- ⌛ Typing animation for interactive experience
- 📱 Responsive and user-friendly layout

---

## 🧠 Approach & Logic
The assistant uses a **keyword-based decision system** to simulate intelligent conversation.

- User input is captured and processed
- The system checks for keywords such as:
  - "process"
  - "timeline"
  - "vote"
  - "eligibility"
  - "india"
- Based on the detected input, the chatbot returns structured responses

This design mimics basic natural language interaction, improving user engagement and accessibility.

---

## ⚙️ How It Works
1. User enters a query or clicks a button  
2. JavaScript processes the input  
3. Matching response is generated  
4. Chatbot displays the reply with a typing effect
